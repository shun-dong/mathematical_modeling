 What is this?
  Author: Jeff O'Neill
  Summary: spectrograms, time-frequency distributions, chirplet analysis, and fractional Fourier transform
  MATLAB Release: R10
  Description: This is a collection of MATLAB files for time-frequency analysis. These programs are either a result of my research or something that I found useful enough to spend the time to implement. Included are a rigorous implementation of time-frequency distributions (Cohen class), some quartic time-frequency distributions, chirplet decomposition based on maximum likelihood estimation, fractional Fourier transform, time-varying filtering, and other useful utilities.

NOTE: The chirplet decomposition requires the Matlab Optimization Toolbox.

For more information, please see http://tfd.sourceforge.net. See also the whale whistle example at http://tfd.sourceforge.net/Whale.
